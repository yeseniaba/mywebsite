www.omega.com/voice/create

Create a voice message in the Omega system

Parameter Data Type Description
country

Default is US

 String Country where the voice message is created. This field only accepts two-character ISO country codes.
home_details
 Object This object includes the following home-related child attributes:
  • city
    (String)
  • zip_code
    (Number)
name

Default is null

 String The name assigned to the voice message. Omega recommends using a descriptive or custom name.

Request Example

curl https://omega.com/voice/create \
-d country="CA" \
-d name="Business Voicemail" \
